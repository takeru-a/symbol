# Symbol

### SetUp
```
# nodeバージョン確認
node -v
npm -v

# ts-node
npm install -g ts-node
tsc -v

# 初期化
npm init -y

# symbol-sdk インストール
npm install symbol-sdk@2 rxjs ws @types/ws

# 実行
ts-node main.ts
```
